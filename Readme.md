## Description
A Multi-Page, API-Driven Cinema Discovery Web App designed to browse movies, TV series, and celebrities. Built using Vanilla JavaScript and Vite, this project strictly adheres to **SOLID principles**, modular architecture, and modern UI/UX standards based on Figma design. The application features a clean, structured UI to discover the latest trending movies, series, and people, with a strong focus on clean software architecture, component reusability, and scalability.

## Features
- **Multi-Page Navigation (SPA Routing):** Seamless transitions between Home, Movies, Series, and Celebrities pages.
- **TMDB API Integration:** Real-time data fetching for trending media, specific genres, and pagination.
- **Dynamic Filtering:** Filter movies and TV shows by specific genres via a sidebar.
- **Pagination:** Navigate through hundreds of movies, series, and celebrities.
- **Responsive Design:** Mobile-first, fully responsive UI following a strict Figma design.
- **Dark/Light Mode:** Toggleable theme with fluid transitions and system preference detection.
- **SOLID Architecture:** Clean, decoupled, and maintainable JavaScript modules.

## Technologies Used
- **HTML5 & CSS3** (Custom Properties, Flexbox, Grid)
- **Vanilla JavaScript** (ES6+, Modules)
- **Vite** (Next-generation frontend tooling)
- **TMDB API** (The Movie Database)

## Usage
- Open the application in your browser.
- Browse the Home page for the latest trending content.
- Navigate to the Movies or Series pages to filter titles by specific genres using the sidebar.
- Use the pagination controls at the bottom of the content lists to browse more items.
- Toggle between light and dark themes using the sun/moon icon in the navigation bar.
